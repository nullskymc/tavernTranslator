欢迎使用 Tavern Translator 角色卡翻译工具！

这是一个专为 SillyTavern 角色卡设计的翻译工具，可以帮助你将英文角色卡翻译成中文。

使用步骤：

  1. 上传 PNG 格式的角色卡文件
  2. 设置翻译参数（需要提供自己的 API 密钥）
  3. 等待翻译完成
  4. 下载翻译后的角色卡文件
